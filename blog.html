What is the Document Object Modal (DOM)?

DOM is a programming inerface for web documents which represents the structuture of a webpage as a tree of objects where each element, like text, images, or buttons is treated as an object. Using Javascript we can interact with these objects and also can change the content or style.

How do you select the element from the DOM?

To select element from the DOM we can use five Javascript methods: 1. getElementById() - Selects an element by its unique ID. 2. getElementsByClassName() - Selects all elements with a specific class name. 3. getElementsByTagName() - Selects all elements with a specific HTML tag. 4. querySelector() - Selects the first element that matches a CSS selector (like class, ID, or tag). 5. querySelectorAll() - Selects all elements that match a CSS selector.
