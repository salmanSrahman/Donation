Important FAQ?

What is the Document Object Modal (DOM)?

DOM is a programming inerface for web documents which represents the structuture of a webpage as a tree of objects where each element, like text, images, or buttons is treated as an object. Using Javascript we can interact with these objects and also can change the content or style.

How do you select the element from the DOM?

To select element from the DOM we can use five Javascript methods: 1. getElementById() - Selects an element by its unique ID. 2. getElementsByClassName() - Selects all elements with a specific class name. 3. getElementsByTagName() - Selects all elements with a specific HTML tag. 4. querySelector() - Selects the first element that matches a CSS selector (like class, ID, or tag). 5. querySelectorAll() - Selects all elements that match a CSS selector.

What is the event delegation in the context of the DOM and why is it useful?

Event deligation is a technique to manage events in a more eficient way in DOM. In this way, instead of using individual event listener for every child elements, we use one event listener in their parent element and it listen individual event from child element. It happened because of JS event bubbling.
Why is it useful? 1. Instead of using multiple event listener, we can use use one event listener for parent element which is less memory use and improved perfomance. 2. If need to add new child elements, then parent event listener will still work for new child element, don't need to add individual event listener for new child element.

How do you manipulate an element's attribute and styles using DOM?

To manipulate an element's attributes we use 3 method. 1. getAttribute() to get the value of an attribute. 2. setAttribute() to update or, edit the value of an attribute. 3. removeAttribute() to remove attribute. To manipulate an element's style we use 3 method. 1. Change a single style. const box = document.getElementById("myBox"); box.style.backgroundColor = "red"; 2. Change multiple styles. box.style.width = "200px"; box.style.height = "200px"; box.style.border = "2px solid black"; 3. Add or remove a class. box.classList.add("new_class"); box.classList.remove("old_class");
